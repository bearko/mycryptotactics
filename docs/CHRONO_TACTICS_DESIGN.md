# クロノ・デッキ（仮）— 歴史偉人 × ローグライク × カードバトル 企画書

## 1. コンセプト

**一文**: *Slay the Spire* 型の**分岐マップ進行**と**デッキ構築ターン制バトル**で、歴史上の偉人をリーダー（キャラクター）として時代を登り切るシングルプレイ作品。

**体験の核**

- **1 ランごとに変わる道筋**: 戦闘・休憩・店・謎イベントのノードを選び、デッキとリソースを育てながらボス層へ向かう。
- **カードは「戦術・演説・奇策」**: 偉人の業績やエピソードをモチーフにした名前・イラスト（立ち絵）で差別化。数値はシンプルに保ち、**キーワード**（脆弱、出血、エナジー増加など）で幅を出す。
- **偉人リーダー**: 開始時に 1 名を選び、**スターター固有カード**と**パッシブ**を持つ（参照データの HP / PHY / INT / AGI をキャラの個性に翻訳してもよい）。

**ビジュアル・アセット方針**

- 偉人立ち絵・UI アイコンは [bearko/mycryptoheroes](https://github.com/bearko/mycryptoheroes) の `Image/` を参照（ライセンスは同リポジトリに従う）。
- 汎用モンスターには `Image/Enemies/` や `Image/Cryptids/` を利用可能。
- モックは `raw.githubusercontent.com` 経由で検証。

## 3. コアゲームループ（ラン内）

1. **マップ**: 層ごとの**ノードグラフ**上を 1 マス進む（分岐・合流あり）。
2. **戦闘**: 毎ターン **エナジー**でカードをプレイ → 敵の**意図（インテント）**を読み **ブロック**・**デバフ**で対処 → ターン終了で手札を捨て、次ターンにドロー。
3. **報酬**: 勝利で**ゴールド**と**3 枚から 1 枚選択**のカード報酬（スキップ可）。
4. **ショップ**: ゴールドでカード削除・カード購入・消耗品。
5. **休憩**: HP 回復または最大 HP 強化のどちらか（定番の二者択一）。
6. **ボス層**: 層の最後で固定またはプールから抽選のボス。撃破で次章／クリア。

死亡でラン終了（メタ進行でアンロックは任意）。

## 4. 主要システム

### 4.1 デッキ・戦闘

- **エナジー**: ターン開始時にリセット（基本 3、カード・パッシブで増減）。
- **ドロー**: ターン開始に一定枚数（例: 5）。山が尽きたら捨て札をシャッフルして山にする。
- **カード種別**: **アタック** / **スキル** / **パワー**（パワーは場に残るバフ・ルール変更）。
- **敵インテント**: 次ターンの行動を列挙（ダメージ数値・デバフ付与など）。ボスは複数行動やフェーズ変化を検討。

### 4.2 マップ・層

- **ノード種別**: 通常戦、エリート、休憩、店、イベント、宝箱（任意）、ランダム、ボス。
- **生成**: シード付き乱数でグラフを生成し、**パス数・接続ルール**で「詰み」を防ぐ（Spire 系の定石）。
- **偉人テーマの層**: 例 — 「戦国回廊」「大航海の港」「革命の街」など、**環境カード**や敵プールを差し替え。

### 4.3 メタ進行（任意）

- クリアで**新リーダー**や**初期レリック**解放。
- 難易度アセンション: 敵強化・報酬減など。

### 4.4 数値・データ

- 偉人マスタは [heroes.json](https://github.com/bearko/mycryptoheroes/blob/main/Data/Heroes/heroes.json) 互換の id・画像パスを保持し、**リーダースキン**として利用。
- バトル UI のパラメータ表示に [BattleIcons](https://github.com/bearko/mycryptoheroes/tree/main/Image/BattleIcons/Parameters) を流用可能。
