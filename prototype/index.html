ヘルプ

ゲームの目的

このプロトタイプは、マップ上でノードを選びながら進むローグライクと、ターン制のカードバトルを組み合わせた体験の検証用です。

ゴールド（GUM）を稼ぎ、デッキにエクステンション由来のカードを加えながら、最終層のボスを倒してランをクリアすることを目指します。

歴史偉人をリーダーに据え、mycryptoheroes のアセットで見た目を構成しています（利用はリポジトリのライセンスに従ってください）。

操作方法

マップ

  • 始まりの塔のマップは下から上へ進みます。接続矢印で次の休憩・店・ボスを読みます。
  • 篝火で HP が回復し、次の層へ進みます。
  • 店では GUM を消費してカードを購入できます。「立ち去る」で次の層へ。

戦闘

  • 手札は扇状に重なったカードです。PC はカードに載せると詳細が開き、クリックでプレイ。スマホは手札エリアで指を置いたカードにフォーカス（白枠）が付き、同じカードの上で指を離すとプレイします。長押しで詳細を出したまま、他のカードへ指を滑らせてフォーカスを切り替えられます。
  • エナジーは手札の上に ⚡ 現在 / 最大 で表示されます。
  • ターン終了は右端。左の山札アイコンから残りデッキ枚数と内訳（引く順は非表示）を確認できます。
  • 敵撃破後はVICTORYが表示されます。少し待ってからもう一度タップすると報酬選択へ進みます（勝利直後の指を離す動作では閉じません）。

その他

  • ラン終了画面のもう一度で最初からやり直せます。
  • このヘルプは Esc でも閉じられます。

パラメーター（画面上の意味）

リーダーと敵のカード下は3行で、ラベルなしのアイコン＋数値だけを表示しています。

  • HP — 0 になると敗北（味方）または撃破（敵）。表示は 現在/最大。
  • PHY — 物理攻撃の基礎ステータス。PHY 依存スキルのダメージに使われます。
  • INT — 知略攻撃の基礎ステータス。INT 依存スキルや回復係数に関わります。
  • AGI — 原作の「チャージ」は使わず、このプロトではクリティカル率に反映されます。
  • 🛡 ガード — ターン中に受ける PHY/INT ダメージを、数値分だけ先に削る軽減層です（味方はターン開始で 0 にリセット）。
  • ✦ シールド — 特殊ダメージ（最大 HP 割合など）だけを先に吸収します。PHY/INT 通常攻撃には効きません。

詳しい原作の定義は MCH ヘルプ：パラメーター を参照してください。

バトルロジック（概要）

数式は My Crypto Heroes 公式ヘルプに沿った簡略実装です。一次ソースへのリンクを併記します。

ダメージとカット率

  • PHY 攻撃を受ける側のカット率: min(40%, floor(相手PHY ÷ 2))
  • INT 攻撃を受ける側のカット率: min(40%, floor(相手INT ÷ 2))
  • PHY/INT 依存の最終ダメージは概ね
    floor(攻撃側ステ × スキル% ÷ 100 × (100 − カット率) ÷ 100)

ヘルプ：ダメージとカット率

クリティカル

一定確率で追加ダメージ。追加分はバフ・デバフを除いた戦闘開始時の基礎 PHY/INTで再計算し、カット率も適用します。

ヘルプ：クリティカル

回復

INT 系の回復では回復係数 (術者INT + 対象PHY) ÷ 2 に、スキルの回復%を掛けた量が HP に入ります。

ヘルプ：パラメーター（回復係数）

シールド

PHY/INT 依存の通常ダメージには無効で、特殊ダメージのみシールドが減り、溢れた分が HP に入ります。

ヘルプ：シールド値
